# 🧠 Convolutional Neural Networks — CMPUT 328

Implementation and exploration of Convolutional Neural Networks (CNNs) built
as part of CMPUT 328 (Visual Recognition) at the University of Alberta.

## 🛠️ Built With

- Python
- PyTorch
- Jupyter Notebook

## 📁 Structure

```
└── CNN.ipynb    # CNN architecture, training, and evaluation
```

## 🚀 Getting Started

1. Clone the repo
2. Install dependencies: `pip install torch torchvision numpy matplotlib`
3. Launch the notebook: `jupyter notebook CNN.ipynb`
